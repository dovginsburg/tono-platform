## Description (≤4000 chars)

Tono is a custom iOS keyboard that helps you say what you mean — in a way that actually lands. Draft a message in any text field, switch to Tono, tap Coach, and see four rewrites plus a risk badge before you send.

**What Tono does**

- **Coach.** Tap the Coach button on any draft to see rewrites.
- **Four axes.** Warmer, clearer, funnier, safer. Each rewrite differs on exactly one axis so you can pick the one that fits the moment.
- **Risk badge.** Low / medium / high, with a one-line explanation of how the message might land.
- **One-tap insert.** Swap the chosen rewrite into the host text field instantly.
- **Style memory.** Rewrites you accept are remembered and used to bias future suggestions toward your voice.
- **Privacy first.** Tono has no server. The keyboard calls your own OpenAI or Anthropic key — your key, your data. Nothing is logged.

**Built for**

- Neurodivergent adults (ADHD, autism-spectrum) navigating social nuance.
- Founders pitching clients or investors.
- Managers giving sensitive feedback in Slack, iMessage, or email.
- Anyone whose texts misfire under stress, anxiety, or when they're tired.

**Why Tono exists**

Most writing tools help you *edit*. Tono helps you *translate*. The pre-send moment — that 8-second window when you're staring at the send button wondering *will this land?* — is the moment nobody owns.

Tono is built on top of a tone-discipline system originally developed for AI agents that write in group chats. The rewrite-axis library is the same one that keeps those agents honest.

**Privacy**

Tono has no servers. Your draft text goes directly from your device to the LLM provider you've configured (OpenAI or Anthropic) using your own API key. Drafts are not logged, sold, or used to train any model. The on-device mock provider is fully offline.

Tono requests Full Access only because Apple's Custom Keyboard API requires it to make network calls. Full Access is the toggle name, not the privacy model — the keyboard does not access the network in password fields, where the OS gives it nothing to read.
